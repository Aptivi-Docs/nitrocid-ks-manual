# v0.0.10.x series

This series was known to have implemented the scripting system in the UESH shell which resulted in the restoration of the commands that were removed earlier, like echo, choice, and beep.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.10.0</summary>

Initial release of this series was made on May 19, 2020 to get to the point and add many features.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added scripting</mark>
* <mark style="color:green;">Restored echo, choice, and beep commands for scripting</mark>
* <mark style="color:green;">Added input command</mark>
* <mark style="color:green;">Added new events</mark>
* <mark style="color:green;">Added color customization to error messages</mark>
