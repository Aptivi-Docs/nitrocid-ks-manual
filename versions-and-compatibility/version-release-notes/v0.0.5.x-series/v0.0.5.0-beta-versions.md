# v0.0.5.0 Beta Versions

This is the first series to have incorporated the version-based beta versions of the upcoming series at its time which started with version 0.0.4.9.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.4.9</summary>

Since version 0.0.4.9, many features were added to the kernel. It included improvements across several areas of the kernel. It was released on July 21, 2018.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added temporary aliases</mark>
* <mark style="color:green;">Added time zones</mark>
* <mark style="color:green;">Added new commands</mark>
* <mark style="color:green;">Added MOTD After Login</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Made Nitrocid KS single-instance</mark>
* <mark style="color:yellow;">Special characters in passwords</mark>
* <mark style="color:yellow;">Better behavior when updating the kernel configuration</mark>
* <mark style="color:yellow;">General improvements</mark>

</details>

<details>

<summary>Version 0.0.4.10</summary>

This is the first version that saw screensavers and mods in the kernel that saw improvements ever since. It was released on August 1, 2018.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added listing computers by name using the lscomp command.</mark>
* <mark style="color:green;">Added screensavers</mark>
* <mark style="color:green;">Added kernel modifications (no command support yet)</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Sysinfo is merged with lsdrivers</mark>
* <mark style="color:yellow;">Improved help definition system</mark>
* <mark style="color:yellow;">Fixed kernel instance checking reporting false positives on reboot</mark>
* <mark style="color:yellow;">General improvements</mark>

</details>

<details>

<summary>Version 0.0.4.11</summary>

Pre-defined aliases were removed as a result of added user-defined aliases in 0.0.4.9. It was out on August 3, 2018.

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Fixed uncolored shell causing input color to be printed as white</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Pre-defined aliases</mark>

</details>

<details>

<summary>Version 0.0.4.12</summary>

This version made a mark on the final beta version of 0.0.5.0. It was out on August 16, 2018.

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Replaced the disco command with the Disco screensaver</mark>

</details>
