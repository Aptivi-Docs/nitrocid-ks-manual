# v0.0.15.x series

This series was out as a third-year anniversary of the kernel to add new awesome features, like the addition of the BedOS, 3Y-Diamond, and TealerOS themes to celebrate the release of the PowerPoint OS collection.

This series was the last series that holds the API v1.2 implementation.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.15.0</summary>

This version celebrated the third-year anniversary of the kernel on February 22, 2021. It added many features and groundbreaking improvements, especially fixing the crash on startup in Linux systems.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added the BedOS, 3Y-Diamond, and TealerOS colors</mark>
* <mark style="color:green;">Added the Irish and Welsh languages</mark>
* <mark style="color:green;">Added new text editor commands, including querychar</mark>
* <mark style="color:green;">Added progress support for notifications</mark>
* <mark style="color:green;">Added text changed indicator to the text editor</mark>
* <mark style="color:green;">Added mod conflict detection</mark>
* <mark style="color:green;">Screensaver timeouts are now customizable except for AptErrorSim, HackUserFromAD, and Dissolve</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Specify the city name in the weather command</mark>
* <mark style="color:yellow;">Fixed crash trying to run maintenance mode</mark>
* <mark style="color:yellow;">Time offsets are much more clear</mark>
* <mark style="color:yellow;">Fixed text editor not opening files after closing the editor for the first time</mark>
* <mark style="color:yellow;">Fixed console output glitches in certain situations</mark>
* <mark style="color:yellow;">Fixed crash on startup in Linux</mark>
* <mark style="color:yellow;">Ping continues even if one device can't be pinged</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

</details>
