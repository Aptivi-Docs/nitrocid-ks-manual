# v0.0.22.x series

<figure><img src="https://officialaptivi.files.wordpress.com/2022/06/ks-ptwo-coreclr.png" alt=""><figcaption></figcaption></figure>

This version series introduced .NET 6.0 support to the kernel, bringing in more performance improvements within the framework. Also, this was the first kernel series that could run on Android phones and tablets.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.22.0</summary>

This version was released on June 12, 2022 to bring in above improvements and add several new miraculous features.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added support for .NET 6.0</mark>
* <mark style="color:green;">Added support for Android</mark>
* <mark style="color:green;">Added dependencies support for mods</mark>
* <mark style="color:green;">Added new commands</mark>
* <mark style="color:green;">Added new screensavers</mark>
* <mark style="color:green;">Added new themes</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Used ReadLine.Reboot to read inputs. This fixes many issues regarding reading inputs on Linux systems.</mark>
* <mark style="color:yellow;">Fixed variety of crashes</mark>
* <mark style="color:yellow;">Removed unnecessary dependency, MadMilkman.Ini, from the main app</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

</details>
