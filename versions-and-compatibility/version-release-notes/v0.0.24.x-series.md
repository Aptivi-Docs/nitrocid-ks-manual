# v0.0.24.x series

<figure><img src="https://officialaptivi.files.wordpress.com/2022/08/ks24-1.png" alt=""><figcaption></figcaption></figure>

This version series was the only series that brought the API version v2.1. It added many features, including the command autocompletion for faster completion, shell presets to make your shells look more beautiful, RetroKS which allows you to run KS 0.0.4.0, and so on.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.24.0</summary>

This version was released on August 2, 2022 to make all the above additions. This version was also touted to have restored the unit converter, this time, more powerful than before.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">RetroKS is back!</mark>
* <mark style="color:green;">Added command autocompletion</mark>
* <mark style="color:green;">Added new splashes (systemd, sysvinit, openrc)</mark>
* <mark style="color:green;">Added support for UESH conditions that require more than three arguments</mark>
* <mark style="color:green;">Added preview to saver settings</mark>
* <mark style="color:green;">Added new tests</mark>
* <mark style="color:green;">Added new screensavers (Indeterminate)</mark>
* <mark style="color:green;">Added new languages (SMS Message, Shakespearean)</mark>
* <mark style="color:green;">Added new themes (Ports from VIM community themes)</mark>
* <mark style="color:green;">Added shell presets</mark>
* <mark style="color:green;">Added new UESH conditions</mark>
* <mark style="color:green;">...and more</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Fixed wrong character for lower left corner in Ramp when the config is being created</mark>
* <mark style="color:yellow;">General API improvements</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">No More POP3 as it's incompatible with ARM</mark>
* <mark style="color:red;">Removed obsolete Mod and Screensaver APIs</mark>
* <mark style="color:red;">Removed report bug prompt</mark>

</details>
