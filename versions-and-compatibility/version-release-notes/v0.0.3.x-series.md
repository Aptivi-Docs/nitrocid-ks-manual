# v0.0.3.x series

A subset of additions and changes to the kernel were made with the release of this series, so we made an exhaustive list of all the 0.0.3.x versions and their changes.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.3.0</summary>

Released on April 30, 2018. It's an even more colorful release thanks to the custom colors feature being added.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">New commands added</mark>
* <mark style="color:green;">New arguments added</mark>
* <mark style="color:green;">New group system (was labeled as permissions)</mark>
* <mark style="color:green;">Custom colors</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Re-written the login system</mark>
* <mark style="color:yellow;">Fixed bugs and made general improvements</mark>

</details>

<details>
