# v0.0.18.x series

This version was purported to be another minor update, although the features that came with it were great.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.18.0</summary>

This version was released on August 2, 2021 to add support for complete string formatting in console writers for more efficient printing, as well as new placeholders and customizable startup banner.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added new placeholders</mark>
* <mark style="color:green;">Added new commands</mark>
* <mark style="color:green;">Added SHA384 support</mark>
* <mark style="color:green;">Added the BeatFader support</mark>
* <mark style="color:green;">Added the Beats Per Minute (BPM) implementation to disco</mark>
* <mark style="color:green;">Added support for complete string formatting in console writers</mark>
* <mark style="color:green;">Added support for output files in sumfile</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">When the help command was invoked on a command that wasn't existent, it shows that the command is not found</mark>
* <mark style="color:yellow;">Flipped the date and time on upper right corner</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

</details>
