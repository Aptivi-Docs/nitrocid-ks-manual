# v0.0.4.x series

This is the first series which was deemed as customizable thanks to the subset of improvements that this version series brought. The below versions were released during the lifetime of the series.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.4.0</summary>

This was the first kernel version that actually got straight to the point by adding the core essentials that were groundbreaking. It first surfaced on May 20, 2018, and it was promised to be the big release.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added settings</mark>
* <mark style="color:green;">Added application command-line and kernel boot arguments</mark>
* <mark style="color:green;">Added the debugging system</mark>
* <mark style="color:green;">Added the writing system</mark>
* <mark style="color:green;">Added themes</mark>
* <mark style="color:green;">Maintenance mode</mark>
* <mark style="color:green;">Calculator and Unit Converter</mark>
* <mark style="color:green;">Added commands</mark>
* <mark style="color:green;">Added an option to show MOTD</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Reduced memory usage</mark>
* <mark style="color:yellow;">General improvements</mark>

</details>

<details>

<summary>Version 0.0.4.1</summary>

Released on May 22, 2018.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Configuration update system in case of kernel updates</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Fixed directory change issues</mark>
* <mark style="color:yellow;">General improvements</mark>

</details>

<details>

<summary>Version 0.0.4.5</summary>

These 0.0.x.5 versions used to be the kernel escrow versions that hosted many additions, much like the major kernel update. It was released on July 15, 2018.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added kernel argument to enable maintenance mode</mark>
* <mark style="color:green;">Time and date widget in the upper right corner</mark>
* <mark style="color:green;">Added more temperature units</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Improvements regarding the welcome banner</mark>
* <mark style="color:yellow;">Improved the help system</mark>
* <mark style="color:yellow;">Fixed bugs</mark>
* <mark style="color:yellow;">General improvements</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">FED (Black and White version of Disco)</mark>
* <mark style="color:red;">Deprecated the prompts</mark>

</details>

<details>

<summary>Version 0.0.4.6</summary>

Released on July 16, 2018.

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Removed what's left from the FED command</mark>
* <mark style="color:yellow;">Pre-defined aliases were deprecated</mark>

</details>

<details>

<summary>Version 0.0.4.7</summary>

Released on July 17, 2018.

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Ping attempts can now be repeated when there is an error</mark>
* <mark style="color:yellow;">General improvements</mark>

</details>

<details>

<summary>Version 0.0.4.8</summary>

Released on April 30, 2021.

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Fixed time and date not showing on reboot</mark>
* <mark style="color:yellow;">Now help displays "no help" when command is not found</mark>
* <mark style="color:yellow;">Fixed help usage showing if the command and arguments are written</mark>

</details>
