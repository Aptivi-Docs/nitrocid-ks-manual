# v0.0.19.x series

This version series was the last series that got released in the year 2021 for API v1.3 to add some more features and improvements. It's also the last series released in the same year.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.19.0</summary>

This version was released on August 24, 2021 that allowed you to manage your mods and debug the built-in screensavers.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added folder download/upload to FTP</mark>
* <mark style="color:green;">Added an option to write full color code in ColorWheel</mark>
* <mark style="color:green;">Added the JSON minifier</mark>
* <mark style="color:green;">Added optional output support for jsonbeautify/jsonminify</mark>
* <mark style="color:green;">Added the Kazakh and Yoruba languages</mark>
* <mark style="color:green;">Added the Linotypo and Typewriter screensavers</mark>
* <mark style="color:green;">Added reading text files support in supported screensavers (only Linotypo for now)</mark>
* <mark style="color:green;">Added the mod manager</mark>
* <mark style="color:green;">Added missed character support in typo screensavers</mark>
* <mark style="color:green;">Added debugging support to built-in screensavers</mark>
* <mark style="color:green;">Restored Turkish language</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Improvements in Settings</mark>
* <mark style="color:yellow;">Fixed cancellation not working properly in some situations</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

</details>
