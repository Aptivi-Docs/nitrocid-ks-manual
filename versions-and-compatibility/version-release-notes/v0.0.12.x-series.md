# v0.0.12.x series

<figure><img src="https://officialaptivi.files.wordpress.com/2022/02/68747470733a2f2f692e696d6775722e636f6d2f4b4f3176314b672e706e67.png" alt=""><figcaption></figcaption></figure>

This kernel version series marks one of the biggest major releases in the history of the first-generation API. It adds so many exciting features and groundbreaking improvements.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.12.0</summary>

This version was out on November 6, 2020. It re-imagined how the kernel looks like, as well as how it functions. It also marked as the era of configuration, because the built-in configuration tool (settings command) was made.

{% hint style="info" %}
The kernel used to have its own separate application for kernel configuration which sported a basic configuration UI. It was deprecated starting from this version.
{% endhint %}

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added the built-in settings tool</mark>
* <mark style="color:green;">Added permanent group support</mark>
* <mark style="color:green;">Added settings support for screensavers</mark>
* <mark style="color:green;">Added BouncingText and Dissolve screensavers</mark>
* <mark style="color:green;">Append mode to output command redirection</mark>
* <mark style="color:green;">Now kernel can reset itself to factory settings by running KS with reset command line argument</mark>
* <mark style="color:green;">Added moving mails</mark>
* <mark style="color:green;">Added the Catalan, Somali, Azerbaijani, Maltese, and Filipino languages</mark>
* <mark style="color:green;">Added the weather functionality that uses OpenWeatherMap to get information</mark>
* <mark style="color:green;">Added FTP, text, and mail shell modding</mark>
* <mark style="color:green;">File deletion and attribute setting</mark>
* <mark style="color:green;">replaceinline text editor command</mark>
* <mark style="color:green;">Your text will be automatically saved when open in the text editor</mark>
* <mark style="color:green;">Added console writers for 24-bit colors</mark>
* <mark style="color:green;">You can now specify multiple files and directories in rm, list, fileinfo, and dirinfo commands</mark>
* <mark style="color:green;">Added blocked devices list in a file</mark>
* <mark style="color:green;">Added support for attachments when sending mail</mark>
* <mark style="color:green;">Dynamic mail detection</mark>
* <mark style="color:green;">Mail encryption/decryption support</mark>
* <mark style="color:green;">Added Debian, NFSHP-Cop, and NFSHP-Racer color themes</mark>
* <mark style="color:green;">Now users can keep connected to their e-mail upon exiting the mail shell</mark>
* <mark style="color:green;">When new mails arrive, the notification is sent</mark>
* <mark style="color:green;">Restored the ping command</mark>
* <mark style="color:green;">...and more</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Updated dependencies</mark>
* <mark style="color:yellow;">Improved script execution</mark>
* <mark style="color:yellow;">Fixed misleading message in list and send commands on mail shell</mark>
* <mark style="color:yellow;">Localizable date and time</mark>
* <mark style="color:yellow;">Made more API-friendly</mark>
* <mark style="color:yellow;">Improved compatibility with mods</mark>
* <mark style="color:yellow;">Events can be adapted with arguments</mark>
* <mark style="color:yellow;">Mods can cooperate with each other</mark>
* <mark style="color:yellow;">Improved kernel update</mark>
* <mark style="color:yellow;">Fixed mail not being able to connect once you disconnect</mark>
* <mark style="color:yellow;">Notifications are now suppressed in screensavers</mark>
* <mark style="color:yellow;">Fixed nasty bug which lets you change chpwd when you're not an administrator.</mark>
* <mark style="color:yellow;">Fixed high CPU usage when remote debugger is on</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Removed command-line arguments for CI as unit tests were implemented</mark>

</details>
