# v0.0.8.x series

This version series was the first to use the 255 colors to make the kernel more colorful and lively. It also made several additions and improvements to the kernel to increase your excitement.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.8.0</summary>

This version was out on February 22, 2020 as a second-year anniversary of the kernel. It made useful and helpful additions and improvements. However, we made an unfortunate decision of adding the voice synthesizer to the kernel.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">255 colors support</mark>
* <mark style="color:green;">Brought back the calculator</mark>
* <mark style="color:green;">Added new commands</mark>
* <mark style="color:green;">Added new languages</mark>
* <mark style="color:green;">Added new screensavers</mark>
* <mark style="color:green;">Added listing partitions and drives</mark>
* <mark style="color:green;">Added beep synth</mark>
* <mark style="color:green;">Added SSH and RPC</mark>
* <mark style="color:green;">Added new placeholders</mark>
* <mark style="color:green;">Added global IPv6 and IPv4 properties</mark>
* <mark style="color:green;">...and more</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Updated dependencies</mark>
* <mark style="color:yellow;">Codepages will change accordingly upon setting language</mark>
* <mark style="color:yellow;">Targeted .NET Framework 4.8</mark>
* <mark style="color:yellow;">Bug and crash fixes</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Removed sses and noaliases</mark>

</details>

<details>

<summary>Version 0.0.8.1</summary>

This version was out as an update to 0.0.8 on March 16, 2020 that made fixes to bugs that affected the base version.

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Updated the FluentFTP and LibVLCSharp libraries</mark>
* <mark style="color:yellow;">FTP command now no longer reports an error if either ftp:// or ftps:// was appended before the host name or address</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Removed the last of sses command</mark>

</details>

<details>

<summary>Version 0.0.8.5</summary>

This version was the last ever version that constituted the major feature addition every 0.0.x.5 release, with the release of this version on March 22, 2020.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Permanent list of users</mark>
* <mark style="color:green;">Added Bengali and Punjabi languages</mark>
* <mark style="color:green;">Added the args command line argument</mark>
* <mark style="color:green;">Added the sumfiles command</mark>
* <mark style="color:green;">Added 24-bit color testing. This was the first sign of 24-bit color support in KS.</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Updated dependencies</mark>
* <mark style="color:yellow;">You can now see passwords as stars to increase security</mark>
* <mark style="color:yellow;">Fixed high CPU usage regarding notifications</mark>
* <mark style="color:yellow;">General improvements</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Useless, unnecessary, and destructive filesystem structure parsing every startup</mark>

</details>
