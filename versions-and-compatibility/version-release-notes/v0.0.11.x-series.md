# v0.0.11.x series

This version series is the last series for API v1.1 based kernels, which started with the release of the initial version, 0.0.11.0.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.11.0</summary>

This version was out on July 25, 2020 to add some exciting features and make some improvements, such as the text editor feature.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added command cancellation</mark>
* <mark style="color:green;">Added sending e-mails</mark>
* <mark style="color:green;">Added text editor</mark>
* <mark style="color:green;">Added mkfile command</mark>
* <mark style="color:green;">Added Serbian language</mark>
* <mark style="color:green;">Added console output redirection support for UESH shell commands</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Fixed port parsing in mail shell</mark>
* <mark style="color:yellow;">Fixed chusrname not taking changes to effect permanently</mark>
* <mark style="color:yellow;">Fixed read command not working with files containing spaces in their names</mark>
* <mark style="color:yellow;">Fixed notifications being shown when dismissing them</mark>
* <mark style="color:yellow;">General fixes and improvements</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Speech synthesizer</mark>
* <mark style="color:red;">Demo account creation</mark>

</details>
