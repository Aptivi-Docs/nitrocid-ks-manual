# v0.0.9.x series

This series was declared to be the smallest major release in the lifetime of Nitrocid KS, which first debuted on April 23, 2020.

***

Here are the versions that were released:

* <mark style="color:green;">IMAP shell added</mark>

* <mark style="color:green;">Added comment for Languages in settings</mark>

* <mark style="color:yellow;">Updated dependencies</mark>
