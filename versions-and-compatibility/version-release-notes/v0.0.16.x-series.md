# v0.0.16.x series

This series gave a massive upgrade to the kernel, causing its API version to increase to v1.3. It added a staggering amount of features and general improvements during four months of non-stop development. It also made the kernel more colorful.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.16.0</summary>

This version did all the above changes to the point that it made a step closer to being a fully-fledged kernel simulator. It made a release on June 12, 2021.

{% hint style="info" %}
On NuGet installs, this kernel version according to the package metadata is 0.0.16.1, although there is no 0.0.16.1.
{% endhint %}

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added new screensavers</mark>
* <mark style="color:green;">GPU probing restored</mark>
* <mark style="color:green;">Dynamic themes</mark>
* <mark style="color:green;">New color themes</mark>
* <mark style="color:green;">New commands</mark>
* <mark style="color:green;">Full naming support to remote debugger</mark>
* <mark style="color:green;">Warning and option color types</mark>
* <mark style="color:green;">Speed dial support to SFTP</mark>
* <mark style="color:green;">New events</mark>
* <mark style="color:green;">New API functions, including new console writers</mark>
* <mark style="color:green;">Anonymous group</mark>
* <mark style="color:green;">New languages</mark>
* <mark style="color:green;">Authentication to SSH by private key</mark>
* <mark style="color:green;">First-user OOBE</mark>
* <mark style="color:green;">You can now cancel executable processes</mark>
* <mark style="color:green;">...and more</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">All filesystem API functions now neutralize the path before performing their operation</mark>
* <mark style="color:yellow;">Configuration files are now made to a JSON format</mark>
* <mark style="color:yellow;">Fixed issues connecting to any FTP server from quickconnect</mark>
* <mark style="color:yellow;">General improvements and bug fixes</mark>

#### <mark style="color:$primary;">Removals</mark>

* <mark style="color:red;">Removed beep synth</mark>
* <mark style="color:red;">Removed debuglog command</mark>

</details>
