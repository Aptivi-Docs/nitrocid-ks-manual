# v0.0.2.x series

Over a month after the first series got released, the kernel earned colorful treats as a major feature. Also, it brought many improvements and some additions. The list was made to fully explain the changes that occurred.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.2.0</summary>

Released on March 31, 2018.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added more commands</mark>
* <mark style="color:green;">Kernel Colors</mark>
* <mark style="color:green;">Basic Internet</mark>
* <mark style="color:green;">Argument System</mark>
* <mark style="color:green;">Password change</mark>
* <mark style="color:green;">Kernel arguments</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Clarified login prompt</mark>
* <mark style="color:yellow;">Removed newlines between command output</mark>
* <mark style="color:yellow;">Fixed background color in Disco and FED</mark>

</details>
