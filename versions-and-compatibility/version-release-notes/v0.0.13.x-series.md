# v0.0.13.x series

This series was a minor update to the API v1.2 implementation to add some features.

***

## <mark style="color:$primary;">Versions</mark>

Here are the versions that were released:

<details>

<summary>Version 0.0.13.0</summary>

This version was out on December 20, 2020 to add SFTP.

#### <mark style="color:$primary;">Additions</mark>

* <mark style="color:green;">Added SFTP</mark>
* <mark style="color:green;">Added SHA512 encryption</mark>
* <mark style="color:green;">Added new text placeholder for host name</mark>
* <mark style="color:green;">Added custom shell prompt style</mark>

#### <mark style="color:$primary;">Improvements</mark>

* <mark style="color:yellow;">Fixed password variable being accessible</mark>
* <mark style="color:yellow;">Fixed console formats in FTP and SFTP</mark>

</details>
